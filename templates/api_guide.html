Get Your Groq API Key

Follow these simple steps to unlock AI-powered math solving

1

Visit Groq Console

Open your web browser and navigate to the Groq Console:

https://console.groq.com

This is where you'll create your free account and manage your API keys.

2

Sign Up or Log In

Create a new account or sign in if you already have one:

  • New User: Click "Sign Up" and use your Google account or email
  • Existing User: Click "Log In" with your credentials
üí° Tip: Using Google sign-in is the fastest way to get started!

3

Generate API Key

Once logged in, follow these steps:

  1. Navigate to "API Keys" in the left sidebar
  2. Click the "Create API Key" button
  3. Give your key a name (e.g., "MathSolver App")
  4. Click "Submit"

  1. Copy your API key
